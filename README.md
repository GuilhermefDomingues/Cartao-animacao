# Credit Card UI

Projeto desenvolvido com HTML e CSS que simula um cartão de crédito moderno com frente e verso. O componente reproduz elementos visuais encontrados em cartões reais, oferecendo uma experiência visual elegante e profissional.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3

## ✨ Funcionalidades

- Cartão frente e verso
- Design inspirado em cartões reais
- Exibição de número do cartão
- Nome do titular
- Data de validade
- Chip do cartão
- Código de segurança (CVV)
- Efeitos visuais modernos

1. Clone ou baixe o projeto.
2. Abra a pasta do projeto.
3. Execute o arquivo `index.html` no navegador.
